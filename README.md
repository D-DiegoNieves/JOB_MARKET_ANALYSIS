# Job Market Analysis - Data Analyst Roles

## Overview
Analyzed 2,252 data analyst job postings scraped from Glassdoor to identify 
the most in-demand skills and highest paying sectors.

## Key Findings
- SQL and Excel are the most requested skills (1,388 and 1,353 job postings)
- Python commands the highest average salary at $74,700 despite being 4th in demand
- Excel pays the least ($71,600) despite being the 2nd most requested skill
- Biotech & Pharma is the highest paying sector for data analysts
- The salary range across all skills is tight (~$5K gap) meaning any of these 
  skills puts you in the same pay band

## Tools Used
Python (pandas, matplotlib, seaborn)

## What This Means for Aspiring Data Analysts
Focus on SQL and Excel for employability, add Python and Tableau for higher pay,
and target Biotech/Pharma roles for maximum salary potential.
